ofro.com.tr

Ofro is an online financial feasibility report genarator tool. It is not just a CRUD web application, but generates reports with graphics based on user inputs.

TECHSTACK

Angular v15 Flask Mongo DB Firebase HTML Bootstrap Git Google Search Chat-GPT

FEATURES

Authentication, Authorization (Angular-Fire) Role-based routes (admin, user) Lazily-loaded components Admin and user guards Interactive form Pandas Dataframes (calculations based on an exsiting excel program) Pie charts and line charts with Mathplotlib Output in .html, .pdf, .xlsx and .docx formats

HOSTING AND DEPLOY

VPS with CentOS Served with Nginx Gunicorn for Flask Git for updates

CHALLENGES

First time deployment experience to a VPS Thousand separator directive was confused my mind :) The time was limited and I didn't write tests. I have converted an excel program into a web-based program, writing all calculating all functions in python from scratch myself. The output has 23 main tables, 5 figures and dynamically selected comments. I have enjoyed but with a stress for time limitation. It works well right now but I need to improve myself about scaling up and database management.
